# Essential Linux Training

This repository contains my notes, command references, and hands-on lab exercises from the **Essential Linux Training Course**.  
It serves as part of my Foundations Phase for building strong Linux and command-line skills — essential for cybersecurity, system administration, and technical development.

---

## Course Overview

The Essential Linux Training course covers:

- Linux concepts and system structure  
- File system navigation and management  
- Shell commands and command-line tools  
- Users, groups, and permissions  
- Processes and system monitoring  
- Scripting basics  
- Networking tools and diagnostics  

---

## 📁 Repository Structure

```

Notes/        – Concept summaries and learning notes

Labs/         – Hands-on exercises and practice tasks

```
