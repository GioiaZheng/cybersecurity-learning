#  TryHackMe — Pre Security Learning Path

A Complete Study Portfolio

This directory contains my full notes, write-ups, and study materials for the **TryHackMe Pre-Security Learning Path**.
The path builds the foundational knowledge required before entering cybersecurity, covering networking, Linux, and security fundamentals.

This repository is organized to serve both as:

*  A personal knowledge base
*  A revision resource
*  A public learning portfolio demonstrating structured technical growth

---

##  Overview of the Learning Path

The TryHackMe Pre-Security path consists of several modules:

### **Networking Fundamentals**

* How the internet works
* IP addressing & subnets
* LAN technologies
* OSI model
* Packets, frames, routing, and protocols

### **Linux Fundamentals**

* Command-line navigation
* File system & permissions
* Processes and system tools
* Shell scripting basics

### **Cyber Security Basics**

* Threats, vulnerabilities, exploits
* Encryption fundamentals
* Authentication & authorization
* Basic security best practices

### **Hands-On Virtual Labs**

Each concept is reinforced through interactive learning and real machines provided by TryHackMe.
